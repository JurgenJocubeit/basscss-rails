# Basscss

Repackaged from <https://github.com/jxnblk/basscss> by Brent Jackson.

## Installation

Add this line to your application's Gemfile:

    gem 'basscss'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install basscss
